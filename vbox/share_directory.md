# Virtual Box における共有フォルダの設定方法
## 動作環境
* ホストOS: Windows 7 Pro 64bit
* ゲストOS: Ubuntu 16.04 64bit

## ホストOS (Windows) 側の設定
1. Virtual Box を起動
2. [設定]→[共有フォルダー]→[共有フォルダーを追加] の順にすすみ、共有したいフォルダを選択
3. [自動マウント]と[永続化]を選択し、OK
以上で完了。

## ゲストOS (Ubuntu) 側の設定
1. 仮想マシンを起動
2. 仮想マシンの上のタブメニューから[設定]→[共有フォルダー]→[共有フォルダーを追加] の順にすすみ、共有したいフォルダを選択
3. [自動マウント]と[永続化]を選択し、OK
4. 端末を立ち上げる
5. mnt/ 以下に共有したいディレクトリを作成
<pre>
$ mkdir /mnt/share
</pre>
6. mount -t vboxsf コマンドで、ホスト OS 側ディレクトリ と ゲスト OS 側ディレクトリを互いにマウントする
<pre>
sudo mount -t vboxsf share /mnt/share
</pre>
以上で完了。

これで、共有ディレクトリにファイルを追加・削除するとその内容が共有される。
